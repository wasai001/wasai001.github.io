How to reset R environment

July 24, 2019
As part of best practice, cleaning the R environment before running the code is suggested. The following code will remove all variables and unload all libraries. # Clear environment of variables and functions rm(list = ls(all = TRUE)) # Clear environmet of packages if(is.null(sessionInfo()$otherPkgs) == FALSE)lapply(paste("package:", names(sessionInfo()$otherPkgs), sep=""), detach, character.only = TRUE, unload = TRUE)

How to count NULLs with SQL and R

July 22, 2019
When representing no values, NULL and NA are used in SQL and R, respectively. Be careful that NULL in R is not the same one as NULL in SQL. Using SQL select count(*) [Number of Null Salaries] from EmP where Salary is NULL Using base R # load the data into dataframe install.packages("RODBC") library(RODBC) conn <- odbcConnect("localhost") Emp <- sqlQuery(conn,"select * from capstone.dbo.Emp") numNAs <- apply(is.na(Emp),2,sum) ratiosNulls <- numNAs/nrow(Emp) ratiosNulls # don't forget to close the connection odbcClose(conn)
