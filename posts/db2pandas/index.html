How to load data from SQL Server to Pandas Dataframe in Python

July 26, 2019
Coding 编程

Here is the way of building connections between SQL Server and Python. Then import data from SQL Server to Pandas Dataframe.

import pyodbc
import pandas as pd
conn = pyodbc.connect('Driver={SQL Server};'
                      'Server=localhost;'
                      'Database=DataBase;'
                      'Trusted_Connection=yes;')

query = "select * from DataBase.dbo.annual_demand"
df = pd.read_sql(query, conn)

conn.close()
