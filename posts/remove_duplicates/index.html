Code - Remove Duplicate Files in the Folder | 编程 - 移除文件夹中的重复文件

July 21, 2019
Coding 编程

This code helps me remove duplicate files, ending with -copy in the folder.

本程序会删除文件夹中以“-copy”结尾的重复文章。

import os
arr = os.listdir()
target = []
for i in arr:
    if "- Copy.JPG" in i or "- Copy.jpg" in i or "- Copy.heic" in i or "- Copy.BMP" in i or "- Copy.PNG" in i or "- Copy.png" in i:
        target.append(i)

for t in target:
    os.remove(t)
